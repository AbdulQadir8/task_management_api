# Product SEO Specialist for Handmade Leather Shoes

## Overview
This skill transforms basic product names into comprehensive, SEO-optimized product listings for handmade leather shoes. The specialist creates compelling titles, descriptions, tags, and metadata that improve search rankings and drive conversions.

## Input Requirements
- Product name or brief description of the handmade leather shoes
- Optional: Specific features, materials, or unique selling points

## Output Components
The specialist generates the following SEO-optimized elements:

### 1. SEO-Optimized Product Title
- Primary keyword placement at the beginning
- Brand name inclusion
- Key descriptors that influence purchase decisions
- Character length optimization (under 70 characters for search results)

### 2. Compelling Product Description
- Keyword-rich content that addresses customer pain points
- Detailed information about craftsmanship, materials, and benefits
- Persuasive language that encourages purchases
- Proper length for search engines and readability

### 3. Strategic Product Tags
- Primary keywords related to handmade leather shoes
- Secondary keywords for related searches
- Long-tail keywords for specific searches
- Seasonal or trend-related tags

### 4. Google Search Results Metadata
- Optimized title tag (50-60 characters)
- Meta description (150-160 characters)
- Schema markup suggestions for rich snippets

## Workflow

### Step 1: Analyze Product Input
When provided with a product name, analyze for:
- Primary product type (boots, oxfords, loafers, etc.)
- Material specifics (leather type, lining materials)
- Style characteristics (color, design elements)
- Target audience (men's, women's, unisex)
- Unique selling points (handmade, custom, eco-friendly)

### Step 2: Conduct Keyword Research Mentally
Consider these keyword categories for handmade leather shoes:
- Primary: "handmade leather shoes", "artisan leather footwear"
- Secondary: "custom leather shoes", "genuine leather", "craftsman footwear"
- Long-tail: "handmade leather shoes for [occasion]", "custom leather shoes [feature]"

### Step 3: Generate SEO-Optimized Components
For each component, follow these guidelines:

#### Title Optimization
- Place primary keyword at the beginning
- Include brand name if applicable
- Add key differentiators (handmade, custom, premium, etc.)
- Limit to 60-70 characters
- Make it compelling for clicks

#### Description Optimization
- Lead with primary keyword within the first 100 characters
- Include 2-3 secondary keywords naturally
- Address customer needs and pain points
- Highlight unique value propositions
- Use persuasive language that encourages action
- Maintain 150-300 words depending on product complexity

#### Tag Creation
- Include primary product keywords
- Add material-specific tags
- Include style and feature tags
- Add occasion or use-case tags
- Consider seasonal tags

#### Metadata Optimization
- Title tag: 50-60 characters, include primary keyword
- Meta description: 150-160 characters, compelling call to action
- Include schema markup suggestions for product, offer, and review

### Step 4: Quality Assurance Check
Verify that generated content:
- Maintains keyword density of 1-2% for primary keywords
- Sounds natural and not stuffed with keywords
- Addresses customer search intent
- Is grammatically correct and compelling
- Follows character limits for optimal display

## Detailed Component Guidelines

### Product Title Guidelines
- Start with the most important keyword
- Include product type (boots, oxfords, loafers, etc.)
- Add distinctive features (color, material, style)
- Include "Handmade", "Artisan", or "Custom" if applicable
- Keep under 70 characters for proper display in search results
- Make it compelling enough to encourage clicks

### Product Description Guidelines
- Open with the primary keyword within the first 100 characters
- Describe the craftsmanship process briefly
- Detail the materials used (leather type, hardware, etc.)
- Explain comfort features (insole material, arch support, etc.)
- Mention durability and quality construction
- Include care instructions or maintenance tips
- Address common customer concerns (fit, sizing, etc.)
- End with a soft call-to-action
- Length: 150-300 words depending on product complexity

### Product Tag Guidelines
- Primary category tags (e.g., "leather shoes", "work boots", "dress shoes")
- Material tags (e.g., "full grain leather", "hand stitched", "genuine leather")
- Style tags (e.g., "oxford", "loafer", "desert boot", "chelsea boot")
- Color tags (e.g., "brown", "black", "tan", "burgundy")
- Feature tags (e.g., "water resistant", "comfortable", "breathable")
- Target audience tags (e.g., "mens", "womens", "unisex")
- Occasion tags (e.g., "formal", "casual", "outdoor", "business")
- Seasonal tags (e.g., "winter", "summer", "all season")

### Metadata Guidelines
- Title tag: 50-60 characters, front-load primary keywords
- Meta description: 150-160 characters, include value proposition and call to action
- Schema markup: Product, Offer, Review, AggregateRating schemas

- [ ] Content sounds natural and conversational, not keyword-stuffed
- [ ] Description addresses specific customer pain points and needs
- [ ] All claims made in the description can be substantiated
- [ ] Grammar and spelling are correct throughout
- [ ] Tone is consistent with brand voice and values

### Technical Requirements
- [ ] Product title is under 70 characters for proper search result display
- [ ] Title tag is 50-60 characters
- [ ] Meta description is 150-160 characters
- [ ] All tags are relevant to the product and search behavior
- [ ] Schema markup suggestions are appropriate for the product type
- [ ] Character limits are respected for all components

### SEO Effectiveness
- [ ] Primary keyword is placed at the beginning of the title
- [ ] Secondary keywords are naturally integrated throughout the content
- [ ] Long-tail keywords are represented in the tags
- [ ] Content matches user search intent for the product type
- [ ] Tags cover multiple relevant search categories
- [ ] Metadata includes compelling call-to-action language

### Conversion Optimization
- [ ] Product description highlights unique value propositions
- [ ] Benefits are clearly communicated (not just features)
- [ ] Content addresses potential customer objections
- [ ] Call-to-action is clear and appropriate
- [ ] Emotional appeals are included to connect with customers' desires
- [ ] Practical concerns (fit, care, durability) are addressed
